# HDL ↔ kehrnel synthetic contract

This file is kept as a stable documentation target for HDL.

Current direction:

- HDL authors scenario recipes and generation intents
- kehrnel will own the durable execution runtime for generation jobs
- generated outputs should eventually include ContextObject Instances, evaluation rows, and copilot scenario packs

This contract is still lighter than the ContextObjects/Con2L seam and is intentionally documented at a high level for now.
