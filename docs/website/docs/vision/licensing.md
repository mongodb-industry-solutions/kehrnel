# Licensing Model

## Runtime code

\{kehrnel\} runtime codebase is distributed under Apache 2.0.

## Strategy documentation/spec artifacts

Strategy documentation and learning artifacts can be published under Creative Commons licenses where explicitly indicated.

## Why this split

This model supports:

- open engineering reuse for runtime implementation
- flexible sharing for strategy knowledge artifacts
- an ecosystem where code and educational assets can evolve with different collaboration needs
