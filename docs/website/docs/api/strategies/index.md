# Strategy Layer API

The strategy layer exposes strategy-specific behavior under:

- `/api/strategies/<domain>/<strategy>/...`

## OpenAPI Docs

- openEHR RPS Dual Swagger: `/docs/strategies/openehr/rps_dual`
- openEHR RPS Dual ReDoc: `/redoc/strategies/openehr/rps_dual`

## Current Strategy APIs

- [openEHR RPS Dual](/docs/api/strategies/openehr/rps-dual)

## Rules For New Strategy APIs

1. Add `docs/website/docs/api/strategies/<domain>/<strategy>/index.md`.
2. Add endpoint-group pages under `docs/website/docs/api/endpoints/` as needed.
3. Link it from this page.
